Bindings présentation

Le bindings consiste à lier des données à des éléments de l'interface utilisateur.

Vous utilisez un modèle (ex:user) et vous souhaitez afficher ses informations dans un formulaire.

il sera alors possible de recuperer l'objet complet ou seulement un champ spécifique.

Il existe plusieurs types de bindings :

  • Property bindings : afficher une valeur
  • Event Binding : gérer les événements
  • Attribute Binding : affichage conditionnel

Exemple de Property Binding

Le Property Binding permet de lier une propriété d'un élément HTML à une propriété d'un composant Angular.

Exemple : <img [src]="sourceImg" alt="texteAlternatifImg">

{{ voyage.destination}}

Du : {{ voyage.dateDepart.toLocaleDateString()}}

Au : {{ voyage.dateRetour.toLocaleDateString()}}

Prix : {{ voyage.prix}}€

Bindings One-Way

Le One-Way Binding permet de lier une donnée à un élément du template, mais sans possibilité de modification.

Exemple :

maValeurString = {{ maValeurString }}

maValeurNumber = {{ maValeurNumber }}

maValeurBoolean = {{ maValeurBoolean }}
