Demo 2 : Les Pipes

Propre à Angular, ils permettent de formatter les données à l'affichage.

Les pipes déjà existants :

Plusieurs pipes existent déjà de base. Afin de pouvoir les utiliser, votre composant aura besoin de les importer soit individuellement via leur nom, soit tous d'un coup avec le CommonModule.


Les pipes string

Les pipes de monnaie

Les pipes de date

Créer ses propres pipes :


Il faudra faire la commande ng g p nomPipe pour créer son propre pipe. Ne pas oublier de l'importer dans les imports du composant pour l'utiliser !
