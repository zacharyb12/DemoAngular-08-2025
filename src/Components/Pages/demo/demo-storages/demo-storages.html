Local Storage Demo

le local storage et le session storage sont des moyens de stocker des données dans le navigateur

Le local storage est persistant, tandis que le session storage est temporaire et expire à la fermeture de l'onglet ou du navigateur.

Le local storage est accessible à partir de n'importe quelle page du même domaine, tandis que le session storage est limité à la session de l'onglet ou de la fenêtre.

Le local storage peut stocker des données jusqu'à 5 Mo, tandis que le session storage peut stocker des données jusqu'à 5 Mo par onglet.

Le local storage est utile pour stocker des données qui doivent persister entre les sessions, comme les préférences utilisateur, tandis que le session storage est utile pour stocker des données temporaires, comme les informations de formulaire.

Le local storage est accessible via l'objet localStorage, tandis que le session storage est accessible via l'objet sessionStorage.

Le local storage et le session storage sont des objets JavaScript qui permettent de stocker des paires clé-valeur dans le navigateur.

Ils sont souvent utilisés pour stocker des données de manière persistante ou temporaire, comme les préférences utilisateur, les informations de session, ou les données de formulaire.
